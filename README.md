# oh-my-termux

Setup Termux with:

 - Nord color scheme
 - Ubuntu Mono with powerline patch
 - fish with agnoster promptline

# Requirement
- curl
```bash
pkg install -y libcurl curl
```

# Installation
```bash
bash -c "$(curl -fsSL https://cdn.jsdelivr.net/gh/4679/oh-my-termux@fish/install.sh)"
```
